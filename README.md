# Facial Recognition System

Sistema de reconocimiento facial para empleados, con interfaz gráfica y flujo de entrenamiento/reconocimiento en vivo.

## Requisitos del sistema
- Python 3.11+
- Dependencias Python de `requirements.txt`

Instala los requisitos de Python con:

```bash
pip install -r requirements.txt
```

## Pruebas
Ejecuta la recolección de pruebas con Pytest para validar que el entorno carga correctamente las dependencias:

```bash
python -m pytest --collect-only
```

> Nota: algunas pruebas requieren acceso a una cámara conectada.
